# RADET Report Generator v2.0

A modern, full-stack web application for generating RADET reports from PostgreSQL databases.

## ✨ Features

- 🎨 **Modern UI** - Beautiful, responsive design with animations
- 🔒 **Secure** - External configuration, no hardcoded credentials
- 📊 **Real-time Preview** - See your query with actual parameters
- 📈 **Report History** - Track all generated reports
- 🔌 **Connection Testing** - Test database connection before saving
- 📁 **CSV Export** - Download reports with timestamped filenames
- ⚡ **Fast & Efficient** - Asynchronous operations with progress indicators
- 🛡️ **Rate Limited** - Protection against abuse

## 🚀 Quick Start
